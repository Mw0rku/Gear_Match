## Design System Notes

### Color Palette
- **Light Mode**: Pink (#EC4899), Sky Blue (#0EA5E9), Amber/Beige (#F59E0B)
- **Dark Mode**: Slate (#1E293B), Purple (#9333EA), Dark variants

### Typography
- Headers: Default system font (no custom font-size classes)
- Body: Default system font (no custom font-weight classes)

### Components
- **Cards**: Rounded-2xl with glassmorphism effect (backdrop-blur)
- **Buttons**: Rounded-full with gradient backgrounds
- **Inputs**: Rounded-2xl with border
- **Icons**: Lucide React icons throughout

### Layout
- **Container**: max-w-sm (mobile-first)
- **Height**: Fixed at 650px
- **Padding**: p-6 standard
- **Spacing**: space-y-3/4/6 for vertical rhythm

### Interactive Elements
- All items have hover states
- Heart icons for favorites
- Star ratings for reviews
- Match percentage badges on recommendations
- Scrollable content with hidden scrollbars
